# Area 面积图生成规则

> 生成或编辑 area 面积图时，必须遵守以下规则。

---

## 规则 1：icon 必须保证语义性

面积图的 icon 以数据点形式显示（position: `top` 或 `bottom`），每个类目对应一个图标。由于面积图常用于**时间序列**或**趋势数据**，icon 的语义选择需要特别注意：

### 场景判断

| 数据类型                             | icon 语义策略            | 搜索方式                                             |
| ------------------------------------ | ------------------------ | ---------------------------------------------------- |
| **时间序列**（如：月度销售额）       | 使用与数据主题相关的图标 | `--keywords`，传入主题关键词                         |
| **多类目对比**（如：各品牌季度数据） | 每个类目需要独立语义图标 | `--per-category-keywords`，按 categoryField 的值搜索 |

### 语义匹配要求

- **国家/地区数据** → 使用国旗图标（`--per-category-keywords` + `--prefer-collection twemoji`）
- **品牌/平台数据** → 使用品牌图标（`--per-category-keywords` + `--prefer-collection mdi`）
- **抽象类目**（如 Q1/Q2 或 1月/2月）→ 使用数据主题图标（如金融类用 `finance`、`chart` 等关键词）

### 注意事项

- 面积图 icon 的 `field` 应指向能区分不同系列/类目的字段
- 如果是时间序列单系列，icon 可能每个时间点都相同 — 此时可考虑只为关键数据点显示 icon 或使用 `annotationPoint` 替代
